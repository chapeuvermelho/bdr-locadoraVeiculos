Software Project: Cadastro de Automóveis para Locadora de Veículos
Version: 1.0.0.0
See also: Software Release
Size: 6.86 MB
Autor: Gilson da Silva
Descrição: 
		Software de exemplo, criado para atender aos requisitos de uma Avaliação Prática para
vaga de Analista Desenvolvedor Java Júnior.

Nesta aplicação é possível cadastrar um veículo, listá-lo em uma grid, 
e efetuar as operações de CRUD em memória.

Campos disponíveis:

Código: Campo de código do registro; é seguencial, podendo o usuário informar um código caso deseje iniciar uma alteração de forma 
manual informando todos os demais campos.

Placa: Campo placa do veículo; com utilização de Máskara 3 letras e 4 digitos.

Marca: Campo Marca, fabricante do veículo.

Modelo: Modelo do veículo.

Combustivel: Tipo de combustível utilizado no veículo.

Cor: Cor do Veículo.

Valor Diária: Valor da locação referente a 1 dia, sendo o mínimo R$ 150,00, não fixado,
apenas para mostrar as validações do campo.
